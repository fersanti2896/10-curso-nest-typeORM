# TypeORM - Postgres

En este repositorio se ven las relaciones, autenticación, autorización y websocket, además de: 

- Uso de `TypeORM`.
- Uso de `Postgres`.
- Creación de `CRUD`.
- Uso de `Constrains`.
- Validaciones.
- Búsquedas.
- Creación de Paginaciones.
- Uso de `DTOs`.
- Uso de `Entities`.
- Uso de `Decoradores` de `TypeORM` para entidades.
- Uso de Métodos `BeforeInsert`, `BeforeUpdate`.
